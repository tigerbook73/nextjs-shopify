# E2E 自动化验收测试

本目录存放项目的自动化验收测试（基于 Playwright）。

主要测试范围包括：

- **核心布局**: 顶栏促销条、导航菜单、底栏。
- **首页与浏览**: 首页 Hero 区域、系列与商品列表。
- **商品详情**: 商品详情页（PDP）、商品图集、SEO 标签、相关商品。
- **购物车**: 购物车侧边抽屉（Cart Drawer）交互。
- **系统状态**: 自定义 404 页面、错误边界恢复、字体配置。

不覆盖真实支付、Shopify 后台配置或第三方外部依赖的完整链路。

## 运行测试

首次本地运行前如缺少浏览器，执行：

```bash
pnpm exec playwright install chromium
```

执行所有测试：

```bash
pnpm exec playwright test
```
